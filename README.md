# 🚧 FixMyRoad

A smart full-stack web application to report and manage potholes and road issues efficiently using real-time tracking and intelligent prioritization.

---

## 🌐 Live Demo
👉 https://fix-my-road-yv2t.onrender.com/

---

## 📌 Problem Statement

Road issues like potholes are often reported multiple times without proper tracking. This leads to inefficiency, redundancy, and delayed resolution.

---

## 💡 Solution

FixMyRoad provides a centralized system where users can report issues, and the system intelligently:

- Detects duplicate complaints within a **20-meter radius**
- Increases report count instead of creating duplicates
- Prioritizes issues using a **dynamic scoring system**

---

## ✨ Key Features

### 👤 User Features
- Report road issues with image and location
- View personal complaint history
- Track status updates

### 🛠️ Admin Features
- View all reported complaints
- Monitor priority-based issues
- Manage complaint status

### 🧠 Smart Features (Unique 🚀)
- 🔁 **Duplicate Detection** (within 20 meters)
- 📊 **Priority Score System**
- Priority = reportCount × severity × trafficLevel
- 📍 Location-based issue mapping
